📱 安装步骤

  • 下载 Mod Manager APK 文件
  • 在手机设置中允许"未知来源"安装
  • 打开 APK 文件进行安装
  • 确保已安装 PojavLauncher
  • 打开 Mod Manager 应用
  • 点击"安装模组"按钮
  • 点击"启动游戏"开始游玩

❓ 常见问题

Q: 为什么安装失败？

A: 请确保在设置中允许"未知来源"安装应用。

Q: 模组不显示怎么办？

A: 确保已安装 Forge 1.20.1-47.2.0 版本，并重启游戏。

Q: 游戏运行卡顿？

A: 降低游戏设置中的渲染距离和图形质量。

Q: 支持哪些设备？

A: 支持 Android 7.0 及以上系统，推荐 4GB+ RAM 的设备。
